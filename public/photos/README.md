Add gallery images in this folder and they will show on the Photos page
automatically.

Supported formats:

```txt
.avif
.gif
.jpeg
.jpg
.png
.webp
```

Photos are sorted by filename, so names like `1-photo.jpg`, `2-photo.jpg`,
`10-photo.jpg` appear in numeric order.

Photo titles can be edited in `photo-titles.json`. Match the key to the image
filename and change the value to the title you want displayed.
